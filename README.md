# Data Analysis Projects

A collection of data analysis projects and SQL practice files.

## Projects

### Layoffs Project
Analysis of layoffs data including data cleaning and exploration.
- `data_cleaning_layoffs.sql` - Data cleaning
- `data_exploration_project.sql` - Data exploration
- `layoffs.csv` - Dataset

## Practice Files
SQL practice files covering various topics:
- Joins, CTEs, Subqueries
- Window Functions, Temp Tables
- String Functions, Case Statements
- Stored Procedures, Events & Triggers

## order_tracker_template
Created an excel order tracking sheet for businesses to track sales and overall performance
Features include:
- Dashboard sheet - KPI cards, orders per platform, total revenue, net profit, etc.
- Order logs sheet - log orders from platforms, automated fee calculation, automated gross/net profit calculation
- Products & Costs sheet - Input catalogue items, platform fees
